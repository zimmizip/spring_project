Comwork
=======
카카오 워크를 벤치마킹한 협업 웹 사이트   
- 기간 : 2020.10.05 ~ 2020.11.05
- 팀원 : 6명   

Technologies
=======
- Java, JavaScript, Ajax
- Spring Framework
- Oracle
- WebSocket(채팅 구현하는데 사용)
- WebRTC(1:1 화상 회의를 구현하는데 사용)   

Start
=====
src/main/webapp/view에 있는 index.jsp를 실행합니다.
